## Get started

1. Install dependencies for the frontend

   ```bash
   cd frontend 
   npm i 
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Install dependencies for the backend 
   ```bash
   cd backend 
   npm i 
   ```
4. Start the server
   ```bash
   npm run 
   node server.js
   ```
